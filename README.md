- **Theme:** Shopify Online Store (customized)

---

## Features Implemented

- ✅ **Homepage** – Store name and welcome message ("Welcome to Stanley Fashion – Style That Speaks for Itself")
- ✅ **Product Pages** – 5 products with images, prices, and descriptions:
  - Classic White Sneakers – RWF 25,000
  - Brown Leather Ankle Boots – RWF 45,000
  - Black High Heel Stilettos – RWF 35,000
  - Black Leather Handbag – RWF 55,000
  - Tan Brown Tote Bag – RWF 40,000
- ✅ **About Us Page** – Store description and mission
- ✅ **Contact Us Page** – Email, phone, location, and contact form
- ✅ **Cart Functionality** – Built-in Shopify add-to-cart and checkout
- ✅ **Navigation Menu** – Links to Home, Catalog, About Us, and Contact Us

## Challenges

- Learning how to navigate and customize the Shopify theme editor without coding experience
- Setting up and publishing pages correctly on the Shopify platform
- Ensuring all navigation links pointed to the correct pages
- Adding a functional contact form within a Shopify page

---

## Lessons Learned

- How to build a fully functional e-commerce store using a no-code platform (Shopify)
- The importance of publishing pages and making them publicly accessible
- How to customize a storefront theme including hero sections, images, and welcome messages
- How navigation menus work and how to link pages correctly
- The value of documentation and version control using GitHub and Markdown
